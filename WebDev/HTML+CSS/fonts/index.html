Exploring different fonts

We can have many different fonts in html. Some of the popular font-family are:-
  • Calibri
  • Courier
  • monospace
  • Times New Roman
You can also add alternate fonts using commas
There's also stuff like font-style:-
  • we can make the text normal
  • we can make the text italic
  • we can make the text oblique
  • we can make the text initial which sets a CSS property to default value
  • we can make the text inherit through which a property is inherited from its parent element (here, ul)

There's also alighnment options in CSS
